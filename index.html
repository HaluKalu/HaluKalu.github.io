Добро пожаловать!

Здесь я размещаю свои тестовые страницы, которые разрабатываются для оттачивания мастерства.

Часть из них являются решениями различных задач с сайтов, некоторые же являются полностью придуманными и написанными мной.

Меню, для эффективности, временно разработано с помощью базовых средств bootstrap. Все остальные же написаны самостоятельно. Часть из них просто CSS, а часть с использованием SCSS.

В меню ссылки выделены разными цветами для удобства.
